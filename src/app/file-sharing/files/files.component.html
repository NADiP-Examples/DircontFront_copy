Правила заполнения шаблона Договора/Визитки
Шаблон создается в формате docx. В шаблоне используется шаблонизатор jinja2.
Для вставки данных из Базы данных используются шаблонные теги:
Пример: {{second_name}} - для вставки Фамилии
Список шаблонных переменных
  • {{value}}
Так же можно использовать логические и прочие теги шаблонизатора jinja2, подробнее в документации шаблонизатора.
